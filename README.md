px2agent
==============

## Usage

```
var px2proj = require('px2agent').createProject('./px_execute.php');
```


## for developers

[![Build Status - master](https://secure.travis-ci.org/tomk79/px2agent.png?branch=master)](https://travis-ci.org/tomk79/px2agent)

### setting up for development

```
$ cd {$project_root}
$ composer install
$ npm install
```

### Test

```
$ npm test
```
